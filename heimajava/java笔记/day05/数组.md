# 数组

## 数组的赋值

使用动态初始化数组时，其中的元素会自动拥有一个默认值，规则如下：

- 整数类型：0；
- 浮点类型：0.0
- 字符类型：'\u0000'
- 布尔类型：false
- 引用类型：null
- 静态初始化其实也有默认值过程，只不过系统自动马上将默认值用大括号内的值进行了替换。

## Java内存的划分

1. **栈**（stack）：存放的都是方法中的局部变量。**方法的运行一定要在栈中。**

   局部变量：方法的参数，或者是方法{}内部的变量

   作用域：{}内的范围；局部变量一旦超出作用域，立刻从栈中消失

2. **堆**（heap）：**凡是new出来的东西都在堆中**

   堆内存里面的东西都有一个十六进制的地址值

   堆内存里面的数据都有个默认值。规则

   ​		整数类型：0

   ​		浮点类型：0.0

   ​		字符类型：'\u0000'

   ​		布尔类型：false

   ​		引用类型：null

3. **方法区**(Method area):存储.class相关信息，包含方法的信息

4. 本地方法栈(Native Method Stack):与操作系统有关

5. 寄存器（Pc register）：与CPU相关
